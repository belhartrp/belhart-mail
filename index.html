Membangun Komunikasi yang Aman dan Bermartabat

Belhart Mail Server menyediakan solusi komunikasi email yang stabil, mudah diakses, dan dapat disesuaikan dengan berbagai kebutuhan pengguna di berbagai tingkat

Belhart Mail Server mendukung komunikasi efektif dan aman untuk keberlanjutan pembelajaran

Belhart

Mail

Proyek ini merupakan bagian dari tugas mata kuliah Administrasi Server. Tujuannya adalah agar mahasiswa memahami secara langsung bagaimana sistem email bekerja, mulai dari instalasi hingga konfigurasi server, serta bagaimana mengelola email secara mandiri menggunakan perangkat lunak open-source.

Sistem Operasi dan Domain

Saya menggunakan Linux Debian sebagai sistem operasi karena dikenal stabil dan banyak digunakan untuk kebutuhan server. Untuk mendukung proyek ini secara nyata, saya juga mengatur domain pribadi, yaitu belhart.net, yang dikonfigurasikan langsung di server Debian yang saya kelola sendiri.

Konfigurasi Postfix

Sebagai Mail Transfer Agent (MTA), saya menggunakan Postfix untuk menangani proses pengiriman dan penerimaan email. Postfix saya konfigurasi agar dapat berjalan melalui subdomain mail.belhart.net, sehingga alamat email yang digunakan dalam pengujian memiliki format seperti nama@mail.belhart.net.


Implementasi Roundcube sebagai Webmail

Agar email dapat diakses melalui browser, saya mengintegrasikan Roundcube sebagai antarmuka webmail. Roundcube menyediakan tampilan yang modern dan responsif, serta mendukung fitur-fitur dasar seperti membaca, mengirim, membalas, dan mengelola email, sehingga mempermudah interaksi dengan mail server yang telah saya bangun.
